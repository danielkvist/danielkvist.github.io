Tania

Tania is a simple and clean theme for Hugo, the framework for building static websites. Even though it’s still on development you can see it working right here, as it is the current theme I am using for my webpage.

It doesn’t use any CSS framework like Bootstrap or Bulma since the goal is to create a theme as light as possible. Only basics CSS properties like CSS Grid and CSS Flexbox.

GitHub

Terrassa

Terrassa is a popular theme for Hugo made to be responsive and accessible. It’s a very classic theme with a top navitation bar (menu that switches depending on the viewport size of the device), a hero with the option to add a CTA button, as many “home” sections as you want and, finally, a contact form that can be used with Netlify forms.

GitHub

Piercer

Piercer is another theme for Hugo. Is very customizable and light and has been designed under the mobile-first philosophy. It offers support for PostCSS and, like the two previous topics, barely uses JavaScript.

Like Terrassa it also provides a contact form with support for Netlify forms.

GitHub

Botio

Botio is one of the projects I’ve worked on the longest. It’s a CLI written in Golang to create and manage easily chatbos for Telegram and Discord. It also has support for different databases like PostgreSQL, SQLite 3 and BoltDB and uses Ristretto for the cache.

But it doesn’t just allow you to create bots connected to a database. It provides an intermediate API that allows you to manage the database through secured endpoints with JWT.

GitHub

Beagle

Beagle is a poweful but simple-to-use CLI also written in Golang to search for an specific username across the Internet.

GitHub

Recommmend

Recommmend is a WebApp built using Golang and Buffalo for the back-end, Bootstrap 4 for the front-end and PostgreSQL for the database that connect to the Spotify API to allow its users to recommend their favorite artists or to receive anonymous recommendations from other users.
