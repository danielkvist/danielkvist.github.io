Tania

Tania is a simple and clean theme for Hugo, the framework for building static websites. It’s still in development but it shows that I know how to use the basics of CSS Grid and CSS Flexbox as well as develop lightweight and responsive web pages without the need for external frameworks.

GitHub

Terrassa

Terrassa is a popular theme for Hugo made to be fast, responsive and accessible. It has a coherent responsive design, a classic navigation menu in large screens which is replaced by a hamburger menu on small screens and a contact form.

GitHub

Piercer

Piercer is another theme for Hugo. Is very customizable but also very light and has been designed under the mobile-first philosophy. It’s color scheme is based on Dracula and the use of JavaScript is minimal. It also has optional support for PostCSS and Service Workers to cache the site.

GitHub

Botio

Botio is a CLI written in Golang to create and manage easily chatbots for Telegram and Discord powered by Cobra. It supports different databases like PostgreSQL, SQLite 3 or BoltDB and uses the Ristretto cache. The authentication between the chatbots and the server that gives access to the database uses JWT.

GitHub

Beagle

Beagle is another CLI written in Golang to search for an specific username across the Internet. It allows you to customize though flags parameters such as: the user agent used to made the requests, the number of goroutines that you want to use or an URL for a proxy.

GitHub

Recommmend

Recommmend is a webapp built using Golang, Buffallo, PostgreSQL, Bootstrap 4 and the Spotify API.

It allows users to recommend their favorite artists or to receive anonymous recommendations from other users.

GitHub
